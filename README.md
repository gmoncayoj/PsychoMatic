**Automated psychometric analysis in R**

PsychoMatic is an R package that streamlines psychometric workflows using SEM-based procedures and literature-informed heuristics. It supports descriptives, EFA, CFA, reliability, measurement invariance, and alignment, with flexible settings and bilingual reporting.
